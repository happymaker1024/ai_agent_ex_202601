 아래 설명한 내용을 구현해줘 
프론트엔드는 react, 백엔드는 fastapi로  구현할거야. 아래의 내용으로 프로그래밍을 해줘
처리 순서 : 프론트 엔드 UI -> 백엔드에서  요청을 받아서 결과를 json 더미데이터로 리턴함 -> 리엑트에서 받아서  레포팅하면 됨

---- 구현 스팩은 다음과 같아 ----
- 프론트엔드 폴더 : frontend/
    - UI 설계 : 첨부파일을 참고해서, UI/UX는 너가 적절히 판단해서 구현해줘
   - backend 요청 API   
        - home :  http://localhost:8000/
        - 레포팅 요청 : http://localhost:8000/invest_report
 - 백엔드 폴더 : backend/
    - 프론트에서 요청한 결과를 더미데이터로 리턴하도록 하면된
    - ai api 요청은 나중에 할 예정이야.
 - "README.md 파일을 업데이트해줘. 프로젝트 설치 방법과 실행 방법을 추가해"
