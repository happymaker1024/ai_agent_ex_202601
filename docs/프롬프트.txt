 아래 설명한 내용을 구현해줘 
프론트엔드는 react, 백엔드는 fastapi로  구현할거야. 아래의 내용으로 프로그래밍을 해줘
처리 순서 : 프론트 엔드 UI -> 백엔드에서  요청을 받아서 결과를 json 더미데이터로 리턴함 -> 리엑트에서 받아서  레포팅하면 됨

---- 구현 스팩은 다음과 같아 ----
- 프론트엔드 폴더 : frontend/
    - UI 설계 : 첨부파일을 참고해서, UI/UX는 너가 적절히 판단해서 구현해줘
   - backend 요청 API   
        - home :  http://localhost:8000/
        - 레포팅 요청 : http://localhost:8000/invest_report
 - 백엔드 폴더 : backend/
    - 프론트에서 요청한 결과를 더미데이터로 리턴하도록 하면된
    - ai api 요청은 나중에 할 예정이야.
 - "README.md 파일을 업데이트해줘. 프로젝트 설치 방법과 실행 방법을 추가해"

# langgraph로 변경하기
현재 폴더에는 crewai를 활용해서 선택한 회사에 투자 보고서를 작성하는 에이전트 서비스가 구현 되어 있어.
프론트엔드의 UI는 변경하지 말고, 백엔드의 처리의 기본 로직도 변경하지 말고, crewai가 했던 역활을 그대로 langgraph로 구현 해줘. 
crewai_/ 폴더 내용을 삭제하면 안됨 

내가 하고 싶은 것은 뭐냐하면, 아래와 같아.
프론트엔드 UI 그대로 두기, 백엔드 기본 처리는 그대로 두고, aigent만 바꿔서 실행해보고 싶은거야.
crewai 또는 langgraph 또는 n8n

지금 너가 할것는 crewai로 구현되어 있는 부분을 langgraph로 구현하는거야.
lagngraph로 에이전트를 구현하는 폴더 :   backend\langgraph_ 생성

내가 쉽게 crewai와 langgraph를 바꿀 수 있도록 해주는 건 허락할게.
